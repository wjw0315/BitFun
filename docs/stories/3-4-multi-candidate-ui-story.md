## 概述

实现多候选技能选择UI，当匹配度相近时展示多个候选供用户选择。

---

## 验收标准

- [ ] 多候选时展示选择提示
- [ ] 展示每个候选的相关度得分
- [ ] 用户可选择执行或取消
- [ ] 支持键盘快捷选择

---

## 详细描述

### UI设计

当存在多个匹配度相近的技能时：
1. 展示候选列表（最多5个）
2. 显示技能名称和描述
3. 显示匹配度得分
4. 支持数字键快速选择

### 交互流程

```
匹配结果 (Top-N)
    ↓
相关度相近? (差值<阈值)
    ↓ [是]
展示候选选择UI
    ↓
用户选择 / 取消
    ↓
执行选中的技能 / 继续对话
```
