## 概述

开发关键词匹配引擎，实现用户输入与可用技能的智能匹配。

---

## 验收标准

- [ ] 输入 "python 排序" 正确匹配 python-patterns
- [ ] 多关键词匹配支持
- [ ] 匹配结果按相关度排序
- [ ] 匹配阈值可配置

---

## 详细描述

### 匹配逻辑

1. **关键词匹配**: 用户输入包含技能的 keywords
2. **触发词匹配**: 用户输入匹配技能的 trigger_patterns
3. **语义相似度**: 计算输入与技能描述的相似度

### 匹配流程

```
用户输入
    ↓
分词处理
    ↓
┌─────────────────────────────────────┐
│ 1. 精确关键词匹配                     │
│ 2. 触发词匹配                        │
│ 3. 语义相似度计算                    │
└─────────────────────────────────────┘
    ↓
排序返回Top-N结果
```
